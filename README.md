## Getting Started
Install dependencies 

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Lint: 

```bash
npm run lint
```

Build static files: 

```bash
npm run build
```

Serve static files: 

```bash
serve ./dist
```

Regenerate routes + gameboard: 

```bash
python ./scripts/parse_svg.py
```
